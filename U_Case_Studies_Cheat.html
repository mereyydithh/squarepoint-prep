7 Universal Failure Modes

  1. Excessive leverage — leverage is a liquidation timer, not a return amplifier
  2. Illusory diversification — ρ → 1 in crisis; "how many INDEPENDENT bets?"
  3. Concentration — single name/factor/trader dominating risk
  4. Model failure / gaming — VaR drops 50% without position change = red flag
  5. Operational / governance — can't manage risk you can't see
  6. Margin / liquidity doom loop — sell → price drops → more margin → more selling
  7. Mechanical amplification — forced flows (rebalancing, rolls, stops) amplify shocks

Key Formulas

Survival test:
$$\text{Survival} = f\left(\frac{\text{Equity cushion}}{\text{Max drawdown path}}\right)$$

Leverage kill math (basis trade):
$$50\text{x leverage} \times 25\text{bp adverse} = 12.5\% \text{ notional loss} \gg 1\text{-}2\% \text{ equity}$$

Concentration check:
- Position as % of float → >10% = dangerous
- Days to exit = position / (ADV × participation rate) → >20 = illiquid

Top 5 Killer Combinations

  1. Leverage + illiquidity + margin call → right at maturity, dead before it
  2. Concentration + past-performance justification → "he's earned it" kills funds
  3. Model trust + assumption rigidity → map ≠ territory
  4. Mechanical selling + ρ → 1 → feedback loop amplifies small shocks
  5. Operational gaps + perverse incentives → can hide losses AND benefit from hiding
