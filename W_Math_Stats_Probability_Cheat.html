Distributions — When to Use

Distribution Use case Key parameter
Normal Portfolio returns, CLT applies μ, σ
Log-normal Prices (can't go <0) μ, σ of log returns
Student-t Fat tails, small samples ν (df): lower = fatter tails
Poisson Event counts (VaR breaches) λ = mean = variance
Exponential Time between events λ = rate; memoryless
GPD Tail modeling (EVT) ξ = shape (>0 = heavy tail)

Bayes' Theorem

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

PM evaluation: Prior 70% skilled → 1 bad month → 44% → 2 bad months → 21%

Hypothesis Testing — PM Alpha

$$t = \frac{\hat{\alpha}}{s / \sqrt{n}}$$

Need t > 2 for significance at 5%.

Years needed to prove skill: $n \geq (z_\alpha / \text{Sharpe})^2$
- Sharpe 0.5 → 15+ years. Sharpe 1.0 → 4 years. Sharpe 2.0 → 1 year.

VaR Backtest Traffic Light (99%, 250 days)

Zone Breaches Verdict
🟢 0-4 OK
🟡 5-9 Questionable
🔴 10+ Broken

Regression Quick Reference

$$r_p = \alpha + \beta_1 F_1 + ... + \beta_k F_k + \epsilon$$

  • R² > 0.80 + insignificant α → closet beta manager
  • Check: VIF (<5), Durbin-Watson (≈2), Adjusted R²

Correlation Traps

  1. ≠ causation
  2. Linear only (use Spearman for non-linear)
  3. Regime-dependent (stock-bond: -0.4 normal → +0.5 inflation)
  4. Time-varying (use rolling windows)

Monte Carlo — When to Use Over Analytical

  • Non-linear portfolios (options/gamma)
  • Path-dependent risk (margin calls, stops)
  • Fat-tailed inputs (t-distribution, GPD)
  • Complex correlations (copulas)

Cholesky: Correlated returns = L × Z (L = Cholesky factor of Σ)

Time Series Quick Reference

φ in AR(1) Behavior
|φ| < 1 Stationary, mean-reverting

φ > 0 Momentum

Half-life: $t_{1/2} = -\ln(2) / \ln(|\phi|)$

Volatility Drag

$$\text{Geometric return} = \text{Arithmetic return} - \frac{1}{2}\sigma^2$$

30% vol → 4.5% annual drag. Over 10yr: 37% less wealth than naive estimate.

√T Scaling Breaks When:

  • Positive autocorrelation (momentum) → VaR HIGHER than √T
  • Volatility clustering → VaR HIGHER
  • Illiquidity (can't exit) → locked in → HIGHER
  • Mean reversion → VaR LOWER than √T
